--------------------------------------------------------------------------------
SpkToolbar
--------------------------------------------------------------------------------

SpkToolbar is a ribbon-like toolbar.

Usage:
  Drop a TSpkToolbar component onto the form
  Double-click it and use the editor window to add tabs, panes and buttons.
